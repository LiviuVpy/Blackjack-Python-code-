## Installation

1. Clone the repository: 
```
https://github.com/LiviuVpy/Blackjack-Python-code-.git
```
2. Change directory into the project folder
3. Create virtual environment: 
```
py -m venv venv
``` 
```
venv/Scripts/activate
```

4. Run the `main.py`
